## 📋 핵심 개념 요약표

| 개념 | 정의 | 자바 예시 | 장 |
|-----|------|----------|---|
| 순수 함수 | 같은 입력 → 같은 출력, 부작용 없음 | `int add(int a, int b)` | 1장 |
| 불변성 | 객체 생성 후 상태 변경 불가 | `record User(...)` | 1장 |
| 참조 투명성 | 표현식을 결과값으로 대체 가능 | `add(2,3)` → `5` | 1장 |
| 일급 함수 | 함수를 값처럼 다룸 | `Function<A,B> f = ...` | 2장 |
| 고차 함수 | 함수를 받거나 반환하는 함수 | `filter`, `map`, `reduce` | 2장 |
| 클로저 | 외부 환경을 캡처한 함수 | 람다에서 외부 변수 캡처 | 2장 |
| 합성 | 작은 함수를 연결해 큰 함수 생성 | `f.andThen(g)` | 3장 |
| 커링 | 다중 인자 함수를 단일 인자 함수 체인으로 | `f(a)(b)(c)` | 3장 |
| 펑터 | map 가능한 컨테이너 | `Optional.map()` | 4장 |
| 모나드 | flatMap 가능한 컨테이너 | `Optional.flatMap()` | 4장 |
| 어플리커티브 | 독립 계산을 결합 | `Validation.combine()` | 4장 |
| 모노이드 | 결합 연산 + 항등원 | `(0, +)`, `("", concat)` | 5장 |
| 곱 타입 | AND 조합 (모든 필드 필수) | `record Person(...)` | 6장 |
| 합 타입 | OR 조합 (하나만 선택) | `sealed interface` | 6장 |
| 지연 평가 | 필요할 때만 계산 | `Stream` | 7장 |
